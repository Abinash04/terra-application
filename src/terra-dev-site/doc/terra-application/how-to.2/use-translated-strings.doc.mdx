# How To Use Translated Strings

Translated strings can be retrieved using the `intl` object provided by `injectIntl` HOC-generator provided by [react-intl](https://github.com/formatjs/react-intl/blob/master/docs/API.md#injectintl).

```jsx
import React from 'react';
import { injectIntl, intlShape } from 'react-intl';

const ExampleComponent = injectIntl(({ intl }) => {
  return (
    <div>
      <p>This component renders a translated string</p>
      <p>{intl.formatMessage({ id: 'test.string.id' })}</p>
    </div>
  );
});

ExampleComponent.propTypes = {
  intl: intlShape,
};

export default ExampleComponent;
```

> Note: The `intl` value is provided by [ApplicationBase](/components/terra-application/terra-application/reference/components/application-base). The version of react-intl required by terra-application (2.9.0) is **not** the latest version.
