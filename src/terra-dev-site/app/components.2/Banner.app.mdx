# Banner

 Banner should be rendered by a component that needs to render a notification with information that needs to be brought to the user's attention. When rendered, the Banner registers itself with the BannerCheckpoint provided by the framework.

## Usage

```jsx
import Banner from 'terra-application/lib/banner';
```
