# Notification Banner

Notification Banner should be rendered by a component that needs to render a notification with information that needs to be brought to the user's attention. When rendered, the Notification Banner registers itself with the NotificationBannerProvider provided by the framework.

## Usage

```jsx
import NotificationBanner from 'terra-application/lib/notification-banner';
```
